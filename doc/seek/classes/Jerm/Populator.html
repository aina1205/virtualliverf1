populate(resource)

adds a resource to the central SEEK archive, referenced by the remote URI, or creates new version if already exists. returns a report: 

 {:response=>:success|:fail|:skipped,:message=>"",:exception=>Exception|nil,:resource=>resource}

[Source]

39:     def populate resource            
40:       begin
41:         if resource.uri.blank?
42:           response={:response=>:fail,:message=>"No URL to data file described"}
43:         elsif !exists?(resource)
44:           response=add_as_new(resource)
